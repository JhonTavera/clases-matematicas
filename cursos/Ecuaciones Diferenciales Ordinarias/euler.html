Calculadora de Euler con redondeo

Ingresa un problema de valor inicial de la forma x' = f(t,x), elige h, el número de iteraciones n y cómo quieres modelar el redondeo. También puedes ingresar una solución exacta opcional para medir errores.

x' = t^2 - x^2, x(0)=2 x' = x - t + 1, x(0)=1 x' = x(1-x), x(0)=0.2 x' = sin(t)-x
Listo para calcular.

Notas

  • Usa ^ para potencias. Ejemplo: t^2 - x^2.
  • Funciones disponibles: sin, cos, tan, exp, log, sqrt, abs y constantes como pi, e.
  • Si ingresas una solución exacta, el programa calcula el error global en los nodos y el error local de truncamiento de cada paso.
  • El redondeo por cifras decimales se modela con una regla decimal simple para visualizar la propagación del error.
